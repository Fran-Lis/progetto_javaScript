Questo progetto è un'applicazione JavaScript che funziona come un counter.

Il valore iniziale è impostato a 0 e può essere incrementato o decrementato cliccando
sui pulsanti + e -.

Oltre ai suddetti pulsanti + e - sono presenti un tasto di reset che riporta il valore a 0
e un campo che permette l'inserimento di un numero di cui incrementare o decrementare il valore del counter.
Se questo campo viene lasciato vuoto, il valore viene incrementato/decrementato di 1.

Quest'ultima caratteristica, oltre a permettere di incrementare/decrementare il valore di numeri diversi da 1, 
apre alla possibilità di utilizzare l'applicazione anche per eseguire addizioni e sottrazioni.

Un'altra caratteristica di questa applicazione è che il container che mostra il valore, come pure i pulsanti e il campo input,
non vengono implementati staticamente dal file html, ma dinamicamente dal file JavaScript.

L'applicazione è dotata di design responsive.

Per aprire l'applicazione è sufficiente aprire
la cartella madre e aprire il file index con il browser desiderato.

Per provare l'app online: https://fran-lis.github.io/progetto_javaScript_Basics/ .
